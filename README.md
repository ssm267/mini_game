# mini_game
بازی مینی گیم متشکل از دو بازی (سنگ،کاغذ،قیچی‌ و دوز) میباشد که در ادامه طرز کار این دو بازی را شرح می دهم.


# بازی اول: سنگ، کاغذ، قیچی:

# 1.قوانین اصلی بازی

سنگ، قیچی را میبرد

قیچی، کاغد را میبرد

کاغذ، سنگ را میبرد

انتخاب یکسان مساوی است.

# 2.روند اجرای بازی


1.شروع بازی

   بازیکن روی یکی از سه دکمه (سنگ، کاغذ یا قیچی) کلیک می‌کند

   2.پردازش جاوا اسکریپت

   دریافت انتخاب کاربر 

   انتخاب تصادفی کامپیوتر
   
   
  3.تشخیص برنده
    

   با استفاده از تابع get winner انتخاب های کاربر با کامپیوتر را مقایسه میکتد
      

 4.به روزرسانی امتیاز
      

  اگر کاربر برنده شود ++userscore
      
  اگر کامپیویتر برنده شود ++compscore

  اگر مساوی شود امتیاز ها تغیییر نمیکند

 5.نمایش نتیجه

 نمایش انتخاب کاربر و کامپیوتر

 نمایش برنده هر دور

 به روزرسانی امتیازات هر دور

# 3.ویژگی های بازی 

  انتخاب تصادفی کامپیوتر
      
   ذخیره امتیاز ها

  بازی مجدد بدون نیاز به رفرش

  پیام های راهنما برای حرکات

# بازی دوم: دوز


# 1.قوانین اصلی بازی

صفحه‌ی 3*3 شامل 9 خانه

به نوبت علامت میگذاریم X , O

هر کس 3 علامت پشت سر هم (سطر، ستون، قطر) بچیند برنده است

اگر همه 9 خانه پر شوند و کسی برنده نشود مساوی است

# 2.روند اجرای بازی

بازی با کامپیوتر:

1.الویت های کامپیوتر

اگر می توانی ببری همان خانه را انتخاب کن

خانه هایی را که باعث برد حریف می شوند را میبندد

مرکز صفحه بازی را بگیرد

گوشه ها را بگیرد

یال ها را بگیرد

بازی دو نفره:

1. ساخت داده ها

ساخت صفحه بازی با ارایه 9 تایی

انتخاب X برای شروع کاربر

2.الگوهای برنده

تعریف تمامی حالات ممکن برای برد

3. حرکت بازیکن

برسی خالی بودن خانه

قرار دادن علامت ها

برسی برنده یا مساوی و تمام بازی

تغییر نوبت

4.پایان بازی

برد: سه علامت پشت سر هم در هر الگو

مساوی: تمام خانه ها پر شده و هیچ برنه ای وجود ندارد

ادامه: هنوز تمامی خانه ها پر نیست و برنده ای مشخص نیست

# 3.ویژگی های بازی

دکمه ریست: بازی را بدون نیاز به رفرش از اول شروع میکند

پیام وضعیت: مشخص میکند نوبت کیست یا چه کسی برنده شده است
